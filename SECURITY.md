# Security Policy

GoldSun is pre-1.0 software. Early builds now include HTTPS-first navigation, fraudulent-site warnings, native WebKit content-rule blocking, tracker-parameter cleanup, optional nonpersistent browsing storage, and Keychain-backed password storage. Password export uses browser-compatible CSV for migration, which is plaintext by design; keep exported files protected and delete them after import elsewhere.

Please still avoid highly sensitive browsing sessions until Chromium extension hosting, release signing, notarization, and signed update verification are complete.

Report security issues privately to the repository owner.
