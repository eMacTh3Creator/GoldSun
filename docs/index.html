Security without browser bloat.

GoldSun keeps the surface area small while tightening the defaults that matter most in daily browsing.

HTTPS first

HTTP navigations upgrade to HTTPS by default, with an optional strict mode for sites that should never fall back.

Tracker cleanup

Known tracking query parameters are removed before navigation, including common ad click and campaign identifiers.

Native content blocking

The WebKit development backend installs a compact rule list for common ads, trackers, and strict-mode noise.

Private storage option

New tabs can use a nonpersistent website data store by default for lower-residue browsing sessions.

Fast by design

GoldSun opens to an offline start page, keeps the UI native, and avoids heavy dependencies in the shell. The result is a browser that feels immediate before the Chromium backend work even lands.

Polished where it counts

Version 0.2.1 adds built-in bookmark and download pages, duplicate-safe bookmarks, favicon-aware tabs and bookmark views, a downloads popover, cleaner toolbar controls, and fuller menu coverage.

Version 0.2.1

The prerelease package installs GoldSun into Applications with a Dock/Finder icon, the GoldSun start page, native bookmarks, built-in bookmark and download pages, tab bar options, a downloads popover, auto update checks, and security-first defaults. It is unsigned until Developer ID signing and notarization are configured.
